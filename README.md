# 🍌 Nano Banana — The Future of Freshness

Welcome to **Nano Banana**, a premium, state-of-the-art interactive product showcase for cold-pressed, 100% natural juices and dairy-free shakes. This web application is meticulously designed to create an immersive, premium user experience that wows visitors at first glance.

---

## 🌟 Highlights & Key Features

* **Glassmorphic Navigation Bar**: A sleek, modern sticky header that adapts elegantly as you browse.
* **Dynamic Theme Engines**: Color palettes, gradients, statistics, features, and pricing details update seamlessly in real time when navigating between flavors.
* **Smooth Video Scrubbing & Transitions**: Immersive visual layouts matching the premium nature of the products.
* **Interactive Live Stats Grid**: Showcases nutritional details with beautifully structured micro-animations.
* **Responsive Layouts**: Designed to look stunning on all screen sizes, from mobile phones to high-resolution desktop monitors.

## 🚀 Running the Project Locally

Since the project uses ES Modules (`import`/`export`), it is recommended to run it through a local development server for local file access and module resolution.

### Method 1: Python HTTP Server (Zero Setup)
Open your terminal in this directory and run:
```bash
python -m http.server 8000
```
Then navigate to `http://localhost:8000` in your web browser.

### Method 2: Node.js (Live Server)
If you have Node.js installed, you can use any quick server:
```bash
npx live-server
```
or
```bash
npm install -g serve
serve .
```
